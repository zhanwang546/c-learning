# c-learning
C语言基础知识点汇总：常量变量、数据类型、数组、循环、指针、排序法
汇总：
一、基础语法
1. 程序结构：#include <stdio.h>（引入标准输入输出头文件，支持printf/scanf等函数）、int main()（程序入口函数，返回int类型表示执行状态）
2. 数据类型：int/char/float/double（基本数据类型，分别对应整数/字符/单精度浮点数/双精度浮点数）、unsigned（无符号修饰，仅存非负数，取值范围更大）、long（长整型，扩大int存储范围）
3. 变量与常量：#define PI 3.14（宏定义常量，编译时直接替换，无类型检查）、const int MAX = 100（const常量，有数据类型，存于内存，不可修改）
4. 输入输出：printf（格式化输出，%d/%f/%c分别对应整数/浮点数/字符）、scanf（格式化输入，需加&取变量地址，避免修改原变量）
5. 运算符：算术（+ - * / %，%仅用于整数）、关系（> < >= <= == !=，比较结果为布尔值）、逻辑（&& || !，短路运算）、自增自减（i++先使用后自增，++i先自增后使用）、赋值（= += -= *= /=，复合赋值简化代码）
二、流程控制
1. 分支结构：if else（单/多分支判断，灵活适配复杂条件）、switch case（多分支等值判断，case后为常量，break跳出分支，default处理默认情况）
2. 循环结构：for（适合已知循环次数，初始化、条件、更新三部分集中）、while（先判断后执行，适合未知循环次数）、do while（先执行后判断，至少执行1次）
3. 跳转语句：break（强制跳出当前循环/switch，终止后续执行）、continue（跳过本次循环剩余语句，直接进入下一次循环）、return（函数返回值，终止函数执行，main中return 0表示正常结束）
三、数组与字符串（重中之重）
1. 一维/二维数组定义与初始化：一维int a[5] = {1,2,3,4,5}（长度固定，下标从0开始）；二维int a[3][3]（行优先存储，可看作一维数组的集合，初始化可分行/连续赋值）
2. 字符串本质：char数组+'\0'（'\0'是字符串结束标志，占1字节，手动添加/系统自动添加，未初始化的字符数组无'\0'时视为普通字符数组）
3. 常用字符串函数：strlen（计算有效字符数，不含'\0'，需含<string.h>头文件）、strcmp（按ASCII码逐字符比较，相等返回0，str1>str2返回正数，反之负数）、strcpy（将src字符串拷贝到dest，包含'\0'，dest需有足够空间）、strcat（将src追加到dest末尾，覆盖dest原'\0'，dest需预留足够空间）
四、函数
1. 函数定义、调用、形参实参：定义格式为返回值类型 函数名(参数列表){函数体}；调用时实参需与形参类型、数量匹配，形参为局部变量，仅在函数内生效
2. 函数声明：在调用函数前声明原型（返回值类型 函数名(参数列表);），解决函数定义在后、调用在前的问题，头文件中常放声明
3. 递归思想：递归公式+结束条件（阶乘：n! = n*(n-1)!，结束条件n=0/1时返回1；斐波那契：F(n)=F(n-1)+F(n-2)，结束条件n=1/2时返回1，递归需避免无终止条件导致栈溢出）
五、指针
1. 基本概念：&取变量地址（获取变量在内存中的存储位置）、*解引用（通过指针访问指向的变量值），例int a=10; int *p=&a;（p存储a的地址，*p等价于a）
2. 指针与数组：a[i] == *(a+i)（数组名本质是首元素地址，通过地址偏移访问元素，二者完全等价，可互换使用）
3. 指针运算：加减（指针+1指向下一个同类型元素，偏移量为类型大小）、比较（同类型指针可比较大小，反映内存地址先后顺序）
4. 空指针：NULL（宏定义的空地址，值为0，指针指向NULL表示不指向任何有效内存，避免野指针）
五、排序法
1.冒泡排序：相邻两两比，大的往后沉；
2.​选择排序：每次挑最小（大），放到最前面；
3.插入排序：像插扑克牌，把新牌插到合适位置；
